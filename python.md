### memo 
#### django 
1. バージョン確認  
``` python3 -m django --version ```  

2. プロジェクト作成  
例) mysiteというプロジェクトを作成したい場合  
``` $ django-admin startproject mysite ``` 

3. 開発用サーバ起動  
``` $ python manage.py runserver ```  

4. アプリケーション作成コマンド 
例) myappというアプリを作成したい場合    
``` $ python manage.py startapp myapp ```
